# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on code and behavior, not individuals.
- Assume good intent, and clarify before escalating.

## Unacceptable Behavior

- Harassment, abuse, or discriminatory language.
- Personal attacks or sustained disruption.
- Sharing private information without permission.

## Enforcement

Project maintainers may remove comments, commits, issues, or contributions that violate this policy.

## Reporting

Report conduct issues privately to the maintainers via repository security contact channels.
